# MP1 Chuck Norris
## Auteurs :
Matteo Belliard et Daniil Belyy
## Description :

Ce projet consiste à rendre hommage au grand, au magnifique, au puissant Chuck Norris !

Le projet est constitué de plusieurs pages html, mises en forme avec un css qui décrivent ce personnage légendaire : sa biographie, le contexte historique de sa carrière et se qu'il a apporté à la pop-culture moderne.
